# SFR Resale Overview Dashboard

## Overview

This Power BI dashboard provides an analysis of Single-Family Residence (SFR) resale data, offering insights into market trends, pricing, and regional performance to assist real estate professionals and investors.

## Objectives

- Analyze resale prices and volumes of single-family residences.
- Identify market trends and regional performance.
- Support data-driven decision-making in real estate investments.
